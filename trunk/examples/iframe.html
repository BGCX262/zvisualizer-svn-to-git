IFrame Shim Tests

This page tests the iframe shim implementation within zVisualizer


Select

The well known IE6 select bug where this windowed element is drawn on top. zVisualizer uses the standard iframe shim technique.



Flash

For a flash object to be in the document flow, it must have a wmode other than "window" in the original source. Otherwise, the default wmode of "window" puts the flash object above everything in the document. Since we don't have access to the source, we can try to use the iframe shim technique, which works on some browsers.

Browser Supported
IE (6/7/8) Yes
Firefox (2/3/3.5) No
Safari (3/4) No
Chrome (1/2) Yes
Opera (9) No
